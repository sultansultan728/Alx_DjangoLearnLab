# LibraryProject

This project was created as part of the **ALX Django Learn Lab** introduction task.

## Project Setup
Steps followed:
1. Installed Django using `pip install django`.
2. Created a Django project using `django-admin startproject LibraryProject`.
3. Ran the development server using `python3 manage.py runserver`.
4. Verified the default Django welcome page at `http://127.0.0.1:8000/`.

## Files Overview
- **manage.py** – Command-line tool to manage this Django project.
- **settings.py** – Project configuration file.
- **urls.py** – URL routing configuration.
- **wsgi.py / asgi.py** – Web server gateway interfaces.
